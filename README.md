## Smart Task Tracker

A modern React-based task management application that allows users to add, delete, and manage tasks efficiently.

### Features
- Add and delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Dark mode toggle 🌙
- Persistent storage using localStorage
- Responsive full-screen UI

### Tech Stack
- React (Hooks)
- JavaScript (ES6+)
- CSS

### Purpose
Built to practice core React concepts like state management, side effects, and component-based architecture.
